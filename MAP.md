# Project Map (MAP.md)

| Folder / File | Role |
| :--- | :--- |
| `src/app/components/` | Main UI orchestration and entry points. |
| `src/app/components/screens/` | Isolated views for each stage of the app. |
| `src/app/hooks/` | **All business logic.** Look here first for "how it works". |
| `src/app/utils/` | Stateless helper functions for recipes and timers. |
| `src/app/lib/` | API / External integration (Recipe AI). |
| `src/types/` | Global TypeScript definitions. |
| `src/app/data/` | Static content and initial state. |
